À Propos

Je m’appelle Léo et je suis étudiant en 1re année de master cybersécurité (CRYPTIS) à l’université de Limoges.

Je possède un intérêt depuis de nombreuses années pour le Web, cela m’a notamment conduit à apprendre les principaux langages du secteur (HTML, CSS, SASS, JavaScript, TypeScript, PHP), ainsi que certains frameworks populaires (Laravel, Symfony, Vue.js). Cette passion ne m’a évidemment pas empêché d’apprendre d’autres langages de programmation tels que C, Java, Kotlin ou Python.

Outre la programmation, je suis également capable de gérer une base de données (MariaDB, MySQL, PostgreSQL), de configurer un réseau et de traiter d’importants volumes de données en utilisant des outils mathématiques avancés.

Pour la partie organisationnelle, j’utilise quotidiennement Git (avec des plateformes associées comme GitHub ou GitLab) et je suis capable de manipuler les principaux diagrammes UML et SysML.
